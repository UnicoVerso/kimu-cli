# Introduction to KIMU-CLI

KIMU-CLI is the official command-line tool for the KIMU framework ecosystem. It allows you to create, manage, and operate KIMU projects, modules, and extensions with simple commands.

## What is KIMU-CLI?
KIMU-CLI provides a unified interface to:
- Scaffold new KIMU applications
- Install and manage modules/extensions
- Build, test, and serve KIMU projects
- Run diagnostics and project info
- Integrate with the KIMU registry and templates

KIMU-CLI is designed for developer productivity, modularity, and ease of use.
